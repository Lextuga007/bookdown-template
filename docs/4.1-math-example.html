4.1 math example

\(p\) is unknown but expected to be around 1/3. Standard error will be approximated

\[ SE = \sqrt(\frac{p(1-p)}{n}) \approx \sqrt{\frac{1/3 (1 - 1/3)} {300}} = 0.027 \]
